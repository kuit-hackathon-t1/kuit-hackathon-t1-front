# KUIT Hackathon T1 Frontend

청춘의 시간을 작은 여행 미션으로 남기는 해커톤 프론트엔드 프로젝트입니다.

사용자는 여행 스타일을 선택하고, 추천받은 미션을 채집해 타임캡슐로 보관할 수 있습니다.  
시도한 미션과 미뤄둔 미션은 나의 섬에 오브젝트로 쌓입니다.

## Tech Stack

- React
- TypeScript
- Vite
- React Router
- TanStack Query
- Zustand
- Tailwind CSS
- Prettier
- ESLint

## Getting Started

```bash
npm install
npm run dev
